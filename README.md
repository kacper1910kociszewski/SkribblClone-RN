# SkribblClone-RN

Real-time collaborative drawing app built with React Native and Socket.io.

## Tech Stack
- **Frontend:** Expo / React Native / SVG
- **Backend:** Node.js / Socket.io
- **Database:** PostgreSQL (In Progress)

## Current Setup Notes
1. Run `npm install` in both `/app` and `/server`.
2. Create a `.env` file in `/server` (see `.env.example`).
3. Start server: `node server.js`
4. Start app: `npx expo start`
